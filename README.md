# Processing Sketches

> *Where pixels come to life, one `draw()` loop at a time.*

A collection of creative coding experiments built with [Processing](https://processing.org/). Think of it as a sketchbook — some pages are doodles, some are full compositions, and at least one is about tuna.

## Sketches

| Sketch | What's going on |
|---|---|
| [FrameRate](./FrameRate/) | Stares at its own frame rate. A little self-absorbed, but honest. |
| [JustDoIt](./JustDoIt/) | Draws a checkmark, step by step. Exports it as a GIF. Motivation sold separately. |
| [Maguro](./Maguro/) | Scrolls the word "Maguro" (🐟 tuna) across the screen. Why? Why not. |
| [Money](./Money/) | Radiating triangles with the word "money" in the center. Art, or a mood board? |
| [SimpleRain](./SimpleRain/) | Perlin-noise rain on a white canvas. 48 lines, zero umbrellas. |
| [Star](./Star/) | A 4×4 grid of randomly colored stars with 5–8 points each. Genuinely festive. |
| [ToAccelerateCircle](./ToAccelerateCircle/) | Expanding concentric circles that speed up as they grow. Things escalate quickly. |
| [ilracer](./ilracer/) | 12 red racers vs. 1 blue underdog on two circular tracks. Chaos, beautifully rendered. |

## Requirements

- [Processing 3+](https://processing.org/download/)
- Some sketches (`Money`, `JustDoIt`) require the [gifAnimation](https://github.com/01010101/GifAnimation) library for GIF export.

## Usage

1. Open any sketch folder in the Processing IDE.
2. Hit the ▶ run button.
3. Watch something happen on screen.

That's it. There's no build step, no `npm install`, no cloud deployment. Just code and pixels.
